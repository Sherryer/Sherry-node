appMain

appMain

appMain适用于适配环境(原系统页面)，例如 base.js 、数据采集、事件回调、资源引用

appMain.back()

  • support 2.0.0

  • summary

    • 调用运行时内部返回
appMain.closeWindow()

  • support 2.0.0

  • summary

    • 关闭浏览器窗口
appMain.getActiveWindow()

  • support 2.0.0

  • Returns contentWindow

  • summary

    • 获取被激活的窗口的 window 对象
appMain.getActiveUrl()

  • support 2.0.0

  • Returns String

  • summary

    • 获取被激活窗口的 URL。
appMain.reloadPage()

  • support 2.0.0

  • summary

    • 刷新当前页面
appMain.openWindow(url)

  • support 2.0.0

  • summary

    • 使用新的窗口打开当前 URL。
appMain.showLoading(text)

  • support 2.0.0

  • summary

    • 显示loading。
appMain.hideLoading(text)

  • support 2.0.0

  • summary

    • 隐藏 loading。
appMain.isAndroid()

  • support 2.0.0

  • Returns Boolean

  • summary

    • 判断是否是安卓系统。
appMain.isIOS()

  • support 2.0.0

  • Returns Boolean

  • summary

    • 判断是否是 IOS 系统。
